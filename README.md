# 🍔 Food Delivery

A modern, responsive food delivery web application built with React. Browse restaurants, select delicious meals, and get them delivered to your doorstep!

**Live Demo:** [frontend-food-delivery-ashen.vercel.app](https://frontend-food-delivery-ashen.vercel.app)

---

## ✨ Features

- 🏪 **Browse Restaurants** - Explore a variety of restaurants and their menus
- 🍽️ **Food Selection** - Easy-to-use interface to select meals and customize orders
- 🛒 **Shopping Cart** - Add/remove items, view order summary
- 💳 **Checkout** - Seamless order placement experience
- 📱 **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- ⚡ **Fast Performance** - Built with Vite for instant page loads
- 🎨 **Modern UI** - Clean and intuitive user interface

---

## 📋 Prerequisites

Before getting started, ensure you have the following installed:
- Node.js (v16 or higher)
- npm or yarn

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Kunal-mob/Food-Delivery.git
cd Food-Delivery
```

### 2. Navigate to Frontend Directory
```bash
cd frontend
```

### 3. Install Dependencies
```bash
npm install
```

### 4. Start Development Server
```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the URL shown in your terminal)

---

## 📦 Available Scripts

In the `frontend` directory, you can run:

### `npm run dev`
Starts the development server with hot module reloading. Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### `npm run build`
Builds the app for production to the `dist` folder. It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run preview`
Serves the production build locally for testing before deployment.

### `npm run lint`
Runs ESLint to check code quality and consistency.

## 🎯 How to Use

1. **Browse Restaurants** - View available restaurants on the home page
2. **Select Items** - Click on a restaurant to view their menu
3. **Add to Cart** - Click "Add to Cart" on your favorite items
4. **Review Order** - Check your cart for items and quantities
5. **Place Order** - Proceed to checkout and complete your order

## 🚢 Deployment

### Deploy to Vercel

The application is already configured for Vercel deployment:

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Import your GitHub repository
4. Vercel will automatically detect Vite configuration
5. Click "Deploy"

**Current Deployment:** https://frontend-food-delivery-ashen.vercel.app
